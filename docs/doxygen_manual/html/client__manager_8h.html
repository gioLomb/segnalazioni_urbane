Descrizione dettagliata

Registry and memory manager for active TCP connections.

Manages the lifecycle of ClientCtx objects through two mechanisms:

  1. Memory: a SlabAllocator provides O(1) alloc/release of contexts.
  2. Tracking: a global doubly-linked list of live connections supports monitoring and ordered graceful shutdown.
