Descrizione dettagliata

High-level wrapper for SQLite3 database operations.

Provides a simplified interface for executing SQL queries and iterating result sets via a cursor. Parameter binding is driven by a compact format string — one character per placeholder:

's' : const char * (string) 'i' : int 'l' : int64_t 'f' : double 'n' : NULL

◆ db_init()

int db_init ( const char *  path)

Opens (or creates) a SQLite database at the given path.

Applies WAL journal mode, NORMAL synchronous level and an 8 MB page cache to the connection before returning.

Precondizione
path points to a writable filesystem location.
Postcondizione
On success a global connection is established and ready for use.
Parametri
pathPath to the .db file.
Restituisce
0 on success, -1 if the database cannot be opened.
