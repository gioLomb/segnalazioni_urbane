Descrizione dettagliata

City geometry module for indexing and validating geographic data.

Manages an internal hash table keyed by city name; callers never touch a Hash_Table pointer directly. Lifecycle: geo_init() at startup, geo_cleanup() at shutdown, geo_lookup() / geo_contains() at any point in between.

◆ geo_init()

int geo_init ( const char *  geojsonPath,
const char *  citiesOut 
)

Loads city geometry from a GeoJSON file into the internal table.

Allocates the hash table, memory-maps the file, parses every feature and computes the bounding box and centroid for each municipality. Optionally writes a JSON array of loaded city names to citiesOut.

Precondizione
geojsonPath is a readable ISTAT GeoJSON file.
Postcondizione
On success the internal table is populated and ready for lookups.
Parametri
geojsonPathPath to the ISTAT GeoJSON file.
citiesOutOptional path to write the city-name JSON array (may be NULL).
Restituisce
Number of cities loaded on success, -1 on fatal error.
