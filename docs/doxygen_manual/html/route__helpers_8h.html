Descrizione dettagliata

Internal helpers shared by route_pages.c and route_api.c.

Not part of the public API — include only from route_*.c files. The public entry point remains handle_request() in route_handler.h.

◆ get_session_user()

bool get_session_user ( const HttpRequest *  req,
User *  u 
)

Extracts and verifies the session cookie from the request.

Reads SESSION_COOKIE_NAME from the Cookie header, then delegates to session_verify() which looks up the token in the session table.

Parametri
reqIncoming HTTP request.
uPopulated with the authenticated user's data on success.
Restituisce
true if a valid session exists, false if missing or expired.
