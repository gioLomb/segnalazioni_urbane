MurmurHash-inspired hash function used by every Hash_Table instance.

Accepts arbitrary binary keys; suitable for strings, integers and structs. The seed (set once at table creation) prevents hash-flooding attacks.

Parametri
keyPointer to the key bytes.
keySizeNumber of bytes to hash.
seedPer-table seed value.
Restituisce
Unsigned long hash value.
