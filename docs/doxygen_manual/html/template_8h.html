Descrizione dettagliata

Lightweight HTML template engine.

Templates are loaded from disk once and stored in an internal pool. Placeholders use {{KEY}} syntax; substitution is driven by a TplVar array.

Usage: tpl_load_files(TPL_LOGIN, TPL_CSS, NULL); const Template *tpl = tpl_get(TPL_LOGIN); tpl_render(tpl, buf, sizeof(buf), vars, nVars);

int tpl_load_files ( const char *  first_path,
  ...

Loads one or more template files into the internal pool.

Each file is memory-mapped, copied into a heap buffer, and pre-parsed into a Fragment array so that tpl_render() needs no runtime scanning. The argument list must be terminated with a NULL sentinel.

Precondizione
Each path must be a readable file; the pool has capacity for up to MAX_TEMPLATES files in total.
Parametri
first_pathPath to the first template file.
...Additional paths, terminated by NULL.
Restituisce
0 if every file loaded successfully, -1 if any file failed.

Renders a template into dest, substituting {{KEY}} placeholders.

Iterates the pre-computed Fragment array: literals are copied directly, variables are resolved via a linear scan of vars. No runtime parsing.

Precondizione
tpl != NULL, dest != NULL, destSize > 0.
Parametri
tplLoaded template to render.
destOutput buffer.
destSizeCapacity of dest in bytes.
varsArray of placeholder substitutions (may be NULL if nVars == 0).
nVarsNumber of elements in vars.
Restituisce
Number of bytes written on success, -1 if dest is too small.
